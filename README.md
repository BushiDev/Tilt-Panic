# Tilt Panic

Minimalistyczna gra mobilna na Androida oparta o sterowanie żyroskopem.

## Gameplay
- Sterowanie: przechył telefonu
- Cel: przetrwać jak najdłużej
- Trudność rośnie z czasem (więcej i szybsze przeszkody)

## Tech
- Unity (2D)
- C#
- Android
